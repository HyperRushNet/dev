Configuration

Pass these parameters into the init() method to control the instance behavior.

ParameterTypeDefaultDescription
containerIdREQUIREDString-Target element ID.
nrREQUIREDNumber-Unique Game ID.
creditsBooleantrueHyperRush watermark.
redirectBlockBooleantrueMutes logs & redirects.
extraPermissionsString""Sandbox flags.

Methods

.init(config)

Bootstraps the runtime, initializes the loader, and mounts the iframe.

// Initialize instance
HyperRush.init({
    containerId: 'game-box',
    nr: 12
});

.destroy(id)

Safe termination: resets the source and clears all animation loops.

// Complete memory cleanup
HyperRush.destroy('game-box');
