embed.min.js

Modern, secure, and ultra-fast game embedding engine.

Overview

embed.min.js is a robust JavaScript solution built to solve the common issues of embedding HTML5 games. It focuses on environment isolation, performance optimization, and top-tier security.

Turbo Engine

Optimizes game loops to a steady 60FPS and freezes execution when the tab is inactive to preserve battery life.

Sandbox Shield

Prevents unwanted redirects and intrusive popups using hard-coded, secure iframe sandbox policies.

Memory Guard

Automatically cleans up event listeners and textures when a game instance is destroyed to prevent memory leaks.

Installation

Include the engine via our global CDN before your closing body tag.

<script src="https://cdn.jsdelivr.net/gh/HyperRushNet/game/1.0.0_embed.min.js"></script>

Quick Start

Define your container and initialize the script with your Game ID.

<!-- 1. Define your container -->
<div id="game-area" style="height: 500px;"></div>

<!-- 2. Initialize HyperRush -->
<script>
HyperRush.init({
    containerId: 'game-area',
    nr: 12,
    redirectBlock: true
});
</script>

Configuration API

Property Type Default Description
containerId String - Target ID of the div container.
nr Number - The unique Game ID from the database.
credits Boolean true Shows the auto-fading HyperRush badge.
redirectBlock Boolean true Enables anti-redirection security.
