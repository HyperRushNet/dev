hrn-game.min.js

The high-performance Web Component for secure, isolated, and lazy-loaded HTML5 gaming.

Quick Start

Integrate the library via JSDelivr CDN and deploy the <hrn-game> component instantly.

<!-- 1. Include the script in your <head> -->
<script src="https://cdn.jsdelivr.net/gh/HyperRushNet/dev/lib/hrn-game_1.2_min.js"></script>

<!-- 2. Use the custom element anywhere -->
<hrn-game nr="17"></hrn-game>

API Attributes

Control the behavior of the sandbox environment using declarative attributes.

Attribute Type Default Description
nr REQ Int - Unique identifier for the game.
credits Bool true Toggles the HyperRush watermark.
redirectblock Bool true Prevents the game from redirecting the top window.
extrapermissions String "" Additional Sandbox flags (e.g. "camera").
provider String "https://hyperrushnet.github.io" Change the start url for the game. (Change provider)

Core Features

I/O Isolation Ensures that game scripts cannot pollute your global window scope or console logs.
Smart Pause Automatically pauses the game runtime via lifecycle hooks when the tab loses focus.
Intersection Loading Saves bandwidth and CPU by only initializing the game when it's visible on screen.
